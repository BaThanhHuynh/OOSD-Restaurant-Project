## ğŸ”§ DETAILED STATE PATTERN IMPLEMENTATION

### StateFactory.java
```java
public class StateFactory {
    private static final Map<DishStatus, Supplier<DishState>> stateMap;
    
    static {
        stateMap = new HashMap<>();
        stateMap.put(DishStatus.ORDERED, OrderedState::new);
        stateMap.put(DishStatus.COOKING, CookingState::new);
        stateMap.put(DishStatus.READY, ReadyState::new);
        stateMap.put(DishStatus.SERVED, ServedState::new);
    }
    
    public static DishState createState(DishStatus status) {
        return stateMap.get(status).get();
    }
}
```

### Concrete State Example
```java
public class OrderedState implements DishState {
    @Override
    public void nextState(OrderItem context) {
        context.setState(new CookingState());
        context.setDishStatus(DishStatus.COOKING);
    }
    
    @Override
    public String getStateName() {
        return "ORDERED";
    }
}
```

```java
@Entity
public class OrderItem {
    @Transient
    private DishState state;
    
    @Enumerated(EnumType.STRING)
```
